# 12. Экран: Профиль

## Назначение

Страница профиля пользователя: настройки, авторизация и экспорт данных. Доступна из основного интерфейса по иконке профиля в шапке; на мобильных — также через вкладку «Профиль» нижней панели вкладок ([15-layout-navigation.md](15-layout-navigation.md)).

## Элементы экрана

### Авторизация

- **Авторизация (скоро)** — раздел или пункт для входа/регистрации. Функциональность планируется к добавлению; на экране может быть заглушка или надпись «Скоро».

### Экспорт данных

- **Экспорт данных в CSV (скоро)** — экспорт записей и всех настроек каждого дела в файл CSV, чтобы пользователь мог:
  - сохранить резервную копию;
  - позже импортировать дело и записи к нему обратно в приложение (импорт — отдельная функция, также «скоро» или в рамках экспорта/импорта).
- Формат экспорта: данные записей + настройки дел (название, описание, блоки, типы, порядок, обязательность и т.д.) в виде, пригодном для последующего импорта.

## Навигация

- Переход на экран профиля — с главного экрана (список дел), через шапку приложения или, на мобильных, через вкладку «Профиль» нижней панели вкладок.
- С профиля пользователь возвращается к списку дел (по вкладке «Дела» на мобильных или по ссылке/кнопке «Назад») или другим разделам.

## Статус функций

- Авторизация: **скоро**.
- Экспорт в CSV и импорт дела с записями: **скоро**.
