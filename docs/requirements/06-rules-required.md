# 06. Обязательность ответов

## Настройка «Обязательное значение»

У каждого блока вопрос-ответ есть свойство **«Обязательное значение»** (да/нет).

- Если «Обязательное значение» включено, при сохранении записи пользователь должен заполнить этот блок (ввести значение или сделать выбор в соответствии с типом блока).
- Если выключено, блок может остаться пустым.

## Поведение при сохранении записи

- Перед сохранением записи выполняется проверка: все блоки с включённым «Обязательное значение» должны быть заполнены.
- Если хотя бы один обязательный блок не заполнен:
  - запись не сохраняется;
  - пользователю показывается сообщение об ошибке (указание на незаполненные обязательные поля или общее сообщение — на усмотрение проектирования).
- После исправления пользователь может снова отправить форму.
