# 01. Глоссарий

## Дело

Форма (карточка), которую пользователь создаёт и многократно заполняет. Содержит эмодзи/символы, название, описание и блоки вопрос-ответ. Один пользователь может иметь множество дел.

## Блок

Один вопрос в рамках дела с заданным типом ввода: число, текст (строка), текст (абзац), один из списка, несколько из списка, шкала, да/нет. У блока есть настройка «Обязательное значение» и возможность менять порядок относительно других блоков.

## Запись

Одна «заполненная форма» — набор ответов по всем блокам дела на момент заполнения. Связана с делом и содержит служебные поля: дата и время добавления, а также технический блок «Заметки».

## Тип блока

Вид вопроса и способ ввода ответа: число, текст (строка), текст (абзац), один из списка, несколько из списка, шкала, да/нет. Подробно описаны в [03-entity-blocks.md](03-entity-blocks.md).
