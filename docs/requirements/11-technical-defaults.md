# 11. Служебные поля (технические свойства по умолчанию)

У каждой записи технически есть три «дефолтных» свойства, общие для всех дел и не настраиваемые пользователем как отдельные типы блоков.

## Блок «Заметки»

- Технический блок, отображаемый **внизу формы** при заполнении и при просмотре записи.
- Свободный текст «на всякий случай» — пользователь может оставить пометку к конкретной записи. Заметки можно редактировать в любое время (при просмотре или редактировании записи).
- Один такой блок на форму: в каждой записи по данному делу есть одно поле «Заметки».

## Дата добавления записи

- При создании записи подставляется текущая дата; дату можно изменить при заполнении формы и отредактировать в любое время (при просмотре или редактировании записи).
- Используется в списке записей и при просмотре одной записи.

## Время добавления записи

- При создании записи подставляется текущее время; время можно изменить при заполнении формы и отредактировать в любое время (при просмотре или редактировании записи).
- Используется в списке записей и при просмотре одной записи.

---

## Итог

Каждая запись всегда содержит: ответы по всем пользовательским блокам дела + поле «Заметки» + дата добавления + время добавления. Дата, время и заметки могут быть отредактированы в любое время (при создании записи — на экране заполнения формы, позже — при просмотре или редактировании записи). В редакторе дела эти служебные свойства не настраиваются — они присутствуют у всех записей по умолчанию.
