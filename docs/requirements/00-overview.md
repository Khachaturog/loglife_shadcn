# 00. Обзор продукта

## Назначение приложения

Log Life — приложение для личных форм и ведения истории записей. Пользователь создаёт «дело» (карточку с формой), заполняет её в нужное ему время и просматривает всю историю добавлений.

## Отличие от Google Forms

- **Google Forms**: форма создаётся для сбора ответов от других людей; ответы приходят от респондентов.
- **Log Life**: форма создаётся для себя; один и тот же пользователь сам заполняет форму когда нужно и видит всю свою историю записей по этому делу.

## Целевой пользователь

Человек, который хочет:
- вести учёт по своим делам (тренировки, привычки, самочувствие, события);
- иметь единую форму с вопросами и заполнять её многократно;
- просматривать историю записей по каждому делу.

## Адаптивность

Приложение адаптивное, **mobile first**: в первую очередь оптимизировано для мобильных устройств. С десктопа заполнение форм и работа с делами также должны быть удобными (корректное отображение, читаемость, удобные элементы ввода).

## Сценарии использования

1. Создать дело (например «Утренняя зарядка») с блоками вопросов (количество приседаний, общее самочувствие по шкале, заметка).
2. В нужное время открыть форму и заполнить ответы — создаётся новая запись.
3. Открыть историю по делу и просмотреть все записи с датой, временем и ответами.
