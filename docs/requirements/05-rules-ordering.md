# 05. Правила упорядочивания

## Порядок блоков вопросов

Блоки вопрос-ответ в рамках одного дела можно передвигать в нужном порядке.

- Реализация: drag-and-drop или кнопки «Вверх»/«Вниз» (на усмотрение проектирования).
- Порядок блоков определяет порядок отображения при создании/редактировании дела, при заполнении формы и при просмотре записи.

## Порядок вариантов в списках

В блоках типов **«Один из списка»** и **«Несколько из списка»** варианты ответов (элементы списка) также можно передвигать в нужном порядке в рамках блока.

- Реализация: drag-and-drop или кнопки перемещения.
- Порядок вариантов определяет порядок отображения при настройке блока и при выборе ответа при заполнении формы.
